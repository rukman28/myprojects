Student Skill Tool

This is a Laravel app which helps students in a single educational institute track their skills. Students can register, select their academic programs, modules, practicals, and skills. They can check if certain skills have already been practiced. The app highlights all the practiced skills in a job description and provides additional information for each program. Students can generate a job-compatible CV with this app.
